# R3Z_Refridgerators
# A Fridge Script for qb and ox inv

# --------------------------------------------------
# R3Z_Refridgerators is a script for FiveM servers running either the qb-core or qbx-core frameworks. The purpose of this script is to provide a realistic refrigerator (or cold-storage) system — e.g., players can store perishables, manage inventory, perhaps interface with food/drink usage or business mechanics.

# ------------------------------------------------------------
# 2. Features

# Here are some of the key features that this script offers:

# Integrates with qb-core/qbx-core’s inventory system.

# Realistic fridge/storage units that players can interact with.

# Option to store perishable items that degrade over time.

# Supports custom fridge locations or business-storage.

# Easy configuration of item lists, storage limits, and cooldowns.

# Compatible with both single-player and multiplayer environments.

# DEPENDENCIES 
#   qb-core
#   ox_lib,
#   ox_target
#   oxmysql
# Optional ox_inv or qb-inv

# ------------------------------------------------------------------

# Commands & Keybinds

# Here are some of the commands built into this script:

# /openfridge — opens the nearest fridge for the player.

# /createfridge [label] — (admin) create a new fridge location.

# /deletefridge [id] — (admin) remove an existing fridge.

# Target the fridge to open
